Changelog

Historique des versions

Toutes les mises à jour, corrections et améliorations de MailGuard.

v2.1.0 Latest Juin 2026
Nouveautés
  • Support dual simultané — Outlook (MSAL) et Gmail (Google OAuth2) actifs en même temps dans la même session
  • Auto-updater embarqué — MailGuard vérifie automatiquement les nouvelles versions au démarrage via GitHub Releases
  • Thread-safety Gmail — Service Gmail isolé par thread (threading.local) pour éviter les conflits en parallèle
  • Backoff exponentiel 429 — L'app gère automatiquement les limites d'API Microsoft et Google sans crash
  • Icône system tray — MailGuard reste actif en arrière-plan dans la barre des tâches Windows
Corrections
  • Résolution du NameError: gmail_email — déclarations global manquantes ajoutées
  • Correction _show_welcome non défini — fonction réinjectée avant sa première utilisation
  • google_credentials.json introuvable au runtime — credentials désormais embarqués dans _GMAIL_CREDS_JSON
  • Vérification bloquante _FGCRD.exists() supprimée de _gmail_login_worker
  • _S.delete timeout non géré — remplacé par le wrapper _api_call() avec retry
  • Taille de police 7.5 rejetée par Tkinter — corrigée en 8 (entiers uniquement)
Améliorations
  • Site web : correction de l'animation fade-up (opacity:0) — les sections s'affichent correctement avec fade-in-anim + fallback JS
  • Compilation PyInstaller optimisée — tous les imports cachés déclarés explicitement pour l'exe Windows
  • Suppression parallèle portée à 52 threads simultanés via ThreadPoolExecutor
v2.0.1 Patch Mai 2026
Corrections
  • Correction du crash au démarrage sur certaines configurations Windows 10 N
  • Timeout de connexion Outlook allongé à 30s pour les connexions lentes
  • Correction du filtre Safe Mode qui ignorait certains sujets en majuscules
  • Suppression définitive non proposée lorsque la corbeille était vide — bouton masqué correctement
Améliorations
  • Amélioration des performances de scan sur les boîtes avec plus de 10 000 emails
  • Interface : affichage du compte connecté dans le header de l'app
v2.0.0 Stable Avril 2026
Nouveautés majeures
  • Refonte complète de l'interface — nouveau design sombre, navigation par onglets
  • Compatibilité Gmail — intégration Google OAuth2 via InstalledAppFlow
  • Système de licence à 3 plans — Particulier / Professionnel / Entreprise
  • Filtre intelligent IA — score de spam 0–100 basé sur sujet, expéditeur et mots-clés
  • Mode corbeille — les emails supprimés passent en corbeille avant suppression définitive optionnelle
  • Safe Mode — protection automatique des emails 2FA, paiements, contrats
  • Scan automatique — analyse en arrière-plan toutes les 5 minutes
Sécurité
  • Authentification 100% OAuth2 — aucun mot de passe stocké par MailGuard
  • Tokens de session chiffrés localement
  • Aucune transmission des emails vers des serveurs tiers
v1.x Legacy 2025

Version initiale — support Outlook uniquement, interface basique, filtre par mots-clés. Non maintenue, mise à jour vers v2.x recommandée.
